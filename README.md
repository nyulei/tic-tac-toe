# Tic Tac Toe (Python + Flask)

A small web-based tic tac toe game served by a minimal Flask app.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Then open http://127.0.0.1:5000/ in your browser.

## Play

- X always starts.
- Click a square to place your mark; first to three in a row wins.
- Scoreboard tracks wins for X, O, and draws; use Reset to start a new round.
